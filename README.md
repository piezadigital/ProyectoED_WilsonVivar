# ProyectoED_WilsonVivar

**Nombre:** Wilson Oswaldo Vivar Solís  

**Descripción del proyecto:** Proyecto de Java para la ejecución y análisis de datos generados para la notación Big O y conocer la tendencia que tienen los algoritmos base utilizados como Factorial, Fibonacci, Búsqueda Lineal y Ordenamiento Burbuja.

## Instrucciones

1. **Compilar**: Requiere Java 11 o superior.  
   - Navegar al directorio raíz del proyecto y ejecutar `javac -d bin src/benchmark/*.java src/algorithms/*.java src/utils/*.java`.

2. **Ejecutar**:  
   - Desde el mismo directorio, ejecutar `java -cp bin benchmark.Main`.

3. **Repositorio**: Este proyecto está alojado en GitHub en https://github.com/piezadigital/ProyectoED_WilsonVivar
